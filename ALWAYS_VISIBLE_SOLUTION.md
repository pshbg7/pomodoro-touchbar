## ‚ùå Why Control Strip Doesn't Work

The "Customize Control Strip" option in System Preferences only shows **system-level items** like:
- Volume
- Brightness  
- Siri
- etc.

Third-party apps cannot add items there for security reasons. Apple restricts this to system apps only.
